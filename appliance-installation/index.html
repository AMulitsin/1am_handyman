Got Questions?

FAQ — Appliance Installation

Do I need to have the appliance delivered first?
+
Yes — we install appliances you already own or have on-site. We do not supply appliances.
Can you remove the old appliance?
+
Yes — disconnection and removal of the old unit is included. Let us know if you need haul-away and we'll arrange it.
Can you connect gas appliances?
+
We connect to existing gas supply lines using approved flexible connectors. We do not run new gas lines — that requires a licensed contractor.
What if the new appliance doesn't fit?
+
We measure and advise before installing. Minor cabinet trimming is sometimes possible — we'll tell you exactly what's involved.
Do you install in apartments and rentals?
+
Yes — we work in apartments, condos, and rentals. We make sure installations are done to code and won't void any warranties.
What brands do you install?
+
Any brand — Samsung, LG, Bosch, GE, KitchenAid, Whirlpool, Viking, and more. Brand doesn't matter; what matters is having the right connections in place.
