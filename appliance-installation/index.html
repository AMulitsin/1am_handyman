Got Questions?

FAQ — Appliance Installation

Do I need to have the appliance delivered first?
+
Yes — we install appliances you already own or have received delivery on. We do not supply appliances.
What if the old appliance needs to be removed?
+
We disconnect and move old appliances out of the way as part of the installation. If you need haul-away, let us know and we'll arrange it.
Can you install gas appliances?
+
We connect appliances to existing gas supply lines using flexible connectors. We do not run new gas lines or do gas pipe work — that requires a licensed contractor.
What if the appliance doesn't fit the existing opening?
+
We measure and advise before installation. If a new appliance doesn't fit, we can often trim cabinetry or modify the space slightly — we'll discuss what's involved.
